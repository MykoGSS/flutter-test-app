# Currency App - Курсы валют Monobank

Flutter приложение для отображения актуальных курсов валют от Monobank.

## Описание

Приложение получает и отображает текущие курсы валют через публичный API Monobank. Показывает курсы покупки/продажи для основных валют (USD, EUR и др.) и кросс-курсы для остальных валютных пар.

## Функции

- ✅ Автоматическая загрузка курсов валют при запуске
- ✅ Обновление данных через pull-to-refresh или кнопку
- ✅ Соблюдение лимитов API (обновление не чаще 1 раза в 5 минут)
- ✅ Отображение времени последнего обновления
- ✅ Обработка ошибок и потери соединения

## API

Используется публичный API Monobank:
- Эндпоинт: `https://api.monobank.ua/bank/currency`
- Аутентификация: не требуется
- Лимиты: данные обновляются на сервере не чаще 1 раза в 5 минут

## Запуск

```bash
flutter pub get
flutter run
```
